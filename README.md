# GOST Digital Signature Lab

Учебный проект по электронной подписи на основе ГОСТ. Скрипт реализует хэширование по `ГОСТ Р 34.11-2012`, операции на эллиптической кривой и интерактивное меню для генерации ключей, подписи файлов и проверки подписи.

## Состав проекта

- `gost_signature_lab.py` - основной скрипт проекта

## Возможности

- вычисление хэша по алгоритму `Стрибог`
- работа со стандартными параметрами кривой
- генерация закрытого и открытого ключей
- подпись файлов
- проверка цифровой подписи
- проверка контрольных примеров из ГОСТ

## Зависимости

Проект использует только стандартную библиотеку Python:

- `os`
- `random`

Установка сторонних библиотек не требуется.

## Запуск

```bash
python3 gost_signature_lab.py
```

## Как использовать

После запуска через меню можно:

1. Просмотреть параметры кривой.
2. Сгенерировать ключевую пару.
3. Подписать файл.
4. Проверить подпись.
5. Проверить реализацию на примерах из стандарта.
